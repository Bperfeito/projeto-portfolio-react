
# 🚀 Portfólio Pessoal | React + Vite

Este é um projeto de portfólio pessoal desenvolvido com *React* e *Vite*, com o objetivo principal de aprendizado e familiarização com a tecnologia.

## 💡 Sobre o Projeto

Este foi meu primeiro contato com o React, onde comecei do zero, sem conhecimento prévio da ferramenta ou do ecossistema.

A ideia foi simples: **aprender construindo**.
A proposta do projeto foi justamente criar algo prático enquanto aprendia os conceitos fundamentais do React e de frameworks modernos de desenvolvimento front-end.

Durante o desenvolvimento, explorei conceitos fundamentais e comecei a entender como aplicações modernas são estruturadas.

## 🧠 O que eu aprendi

* Estrutura básica de aplicações com React
* Componentização e reutilização de código
* Organização de pastas e boas práticas iniciais
* Uso do Vite para otimização do ambiente de desenvolvimento
* Fundamentos de JavaScript aplicados ao React

## 🛠️ Tecnologias

* React
* Vite
* JavaScript
* HTML5
* CSS3

## ⚙️ Como executar

```bash
# Clone o repositório
git clone <seu-repositorio>

# Entre na pasta
cd <nome-do-projeto>

# Instale as dependências
```
