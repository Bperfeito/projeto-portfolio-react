# Portfolio React - Guia de inicio

Este projeto foi criado para te ajudar a aprender React construindo algo util desde o primeiro dia: o seu proprio portfolio.

## O que estamos usando

- React: biblioteca para criar interfaces em componentes.
- Vite: ferramenta que cria e roda o projeto com muita rapidez.
- JavaScript: linguagem principal do projeto.
- CSS: onde estilizamos a interface e deixamos tudo responsivo.

## Como rodar o projeto

1. Abra o terminal na pasta do projeto.
2. Rode `npm run dev`.
3. Abra no navegador o endereco que o terminal mostrar.

## Arquivos mais importantes

- `src/main.jsx`: ponto de entrada da aplicacao.
- `src/App.jsx`: estrutura principal da pagina.
- `src/data/portfolioData.js`: seus textos, links e dados pessoais.
- `src/App.css`: estilos da interface.
- `src/index.css`: estilos globais do projeto.

## Como estudar sem se perder

1. Rode o projeto e veja a tela pronta.
2. Abra `src/data/portfolioData.js` e troque seu nome, links e descricoes.
3. Depois leia `src/App.jsx` e localize na tela cada secao que aparece no codigo.
4. Por fim, explore `src/App.css` para entender como o layout foi montado.

## Conceitos de React que este projeto ja te mostra

- JSX
- Componentes
- Importacao de arquivos
- Renderizacao de listas com `map()`
- Separacao entre dados e interface
- Estrutura de projeto com Vite

## Comandos uteis

- `npm run dev`: inicia o ambiente de desenvolvimento.
- `npm run build`: gera a versao de producao.
- `npm run lint`: verifica problemas de padrao no codigo.

## Proximo passo sugerido

Depois de personalizar os dados, a melhor evolucao para estudo e:

1. Criar uma nova secao no `App.jsx`.
2. Adicionar um projeto real seu na lista `projects`.
3. Fazer deploy gratuito em uma plataforma como Vercel ou Netlify.
